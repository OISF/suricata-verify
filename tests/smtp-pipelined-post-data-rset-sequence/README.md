# Description

SMTP pipelining regression where the client sends `RSET`, a second envelope,
and `DATA` before the server replies to the previous DATA terminator.

# PCAP

The pcap is generated by the email sandbox scenario
`pipelined-post-data-rset-sequence` in `smtp-oddseq.py`.

The test checks that both completed message transactions are logged with their
SMTP envelope fields and parsed email headers.
